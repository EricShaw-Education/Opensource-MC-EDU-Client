# Minecraft EDU Client (Open Source)

## About

Minecraft EDU Client is a free and open-source educational Minecraft client designed for learning, experimentation, and community-driven development.

The project can be downloaded and used completely free of charge.

## Getting Started

1. Download or clone the repository.
2. Open and run `Mc-EDU-Launcher-official.py`.
3. Create a user account within the launcher.
4. Log in using your account credentials.
5. Press the **"Launch Minecraft EDU"** button.
6. Wait for the game to load. Initial startup may take some time depending on your system.

## Supported Languages

The launcher currently supports the following languages:

* English
* German
* French

Additional languages may be added in future updates.

## Multiplayer Information

Please note that this client **cannot connect to official Minecraft servers**.

Multiplayer is only available through servers specifically created and configured for Minecraft EDU Client.
